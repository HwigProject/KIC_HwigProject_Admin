# 휙_프로젝트(식품 쇼핑몰) 관리자 레포지토리

### 내용(SpringMVC, MyBatis를 이용한 기능 구현)
 - 관리자 페이지 구현
 - 메인 페이지의 요청을 응답하는 기능

### 역할분담
 - 지나)
   - 판매자, 회원 관리
   - 관리자 페이지 로그인(Spring Security 사용)
   - 주문 관리 
 - 지혜)
   - 판매자 상품 등록 및 관리
   - 관리자 상품 승인 및 관리
 - 원섭)
   - 공지사항
   - 이벤트
   - FAQ
   - 1대1 문의
   - 메인페이지 로그인
   - 고객후기
  
### 서버
 - AWS를 이용함
 - ubuntu 16.04 버전
 - DB서버는 오라클 사용
